# Fellowship Character Inspiration Folders

These folders hold **personal reference images only**. They are NOT referenced by the Bible app and will NEVER appear on the site.

## How this works

- Drop inspiration images into the appropriate character folder (e.g. `temgar/` for Temgar's look references).
- The Bible's Character Gallery reads from `src/data/characters.js`. Nothing in `inspiration/` folders is ever pointed to by the data file, so these images stay invisible to the site.
- When you are ready to design a custom portrait for a character, use the images in this folder as prompt inspiration — then generate your own original art elsewhere and place it in the fellowship folder as the final portrait.

- `temgar/` — Temgar, the Iron Spear — Mongolian-Viking, braids, beads, ice-blue eyes
