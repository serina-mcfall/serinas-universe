# Fen Sect Character Inspiration Folders

These folders hold **personal reference images only**. They are NOT referenced by the Bible app and will NEVER appear on the site.

## How this works

- Drop inspiration images into the appropriate character folder (e.g. `jing-huo/` for Jìng Huǒ's weapon references).
- The Bible's Armoury tab reads from `src/data/weapons.js`. Nothing in `inspiration/` folders is ever pointed to by the data file, so these images stay invisible to the site.
- When you are ready to design a custom card for a weapon, use the images in this folder as prompt inspiration — then generate your own original art elsewhere and place it in the weapon's final location.

- `meng-huo/` — Měng Huǒ (Fierce Fire) — youngest — Forge discipline (gauntlets)
- `chaos-quintet/` — The Chaos Quintet (next gen)
