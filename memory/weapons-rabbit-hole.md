## THREE OPTIONS I OFFERED FOR CARD IMPLEMENTATION

**Option B — App-rendered from data (RECOMMENDED).** Each weapon is structured data (name, image, labels array, description, cost). React component draws the card automatically. Pro: fast, consistent, can replace with hand-designs later. Con: more standardised.
